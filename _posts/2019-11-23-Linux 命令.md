#### Linux 向文件末尾追加命令

```
//echo后边用单引号包围要添加的内容
echo 'add content'>>/home/data/test.sh
注意：>> 是追加
```

```
echo 'add content'>/home/data/test.sh
注意：> 是删除原有内容，添加
```

#### Mac显示隐藏文件

```
defaults write com.apple.finder AppleShowAllFiles -boolean true
killall Finder
```

#### 关闭显示隐藏文件

```
defaults write com.apple.finder AppleShowAllFiles -boolean false
killall Finder
```
